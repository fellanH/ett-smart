<task type="auto">
  <name>Task 1: Create validation module with file and cell utilities</name>
  <files>webapp/validation.py</files>
  <action>
Create new validation.py module with these functions:

1. **sanitize_cell(value)** - Prevent CSV formula injection
   - If value starts with `=`, `+`, `-`, `@`, or tab character, prefix with single quote `'`
   - Exception: Do NOT sanitize URLs starting with "http" (keep Allabolag links clickable)
   - Handle None and non-string values gracefully
   - Return sanitized string

2. **detect_encoding(file_bytes, sample_size=10000)** - Detect file encoding
   - Use charset-normalizer library
   - Sample first 10KB for performance
   - Return detected encoding string (default to 'utf-8' if detection fails)
   - Handle empty files gracefully

3. **validate_csv_file(uploaded_file)** - Validate uploaded CSV before processing
   - Returns dict with keys: valid (bool), error (str|None), warnings (list), df (DataFrame|None)
   - Checks:
     - File is not empty (0 bytes) -> error: "File is empty"
     - File can be parsed as CSV -> error: "Invalid CSV format: {details}"
     - File has data rows (not just headers) -> error: "File contains only headers, no data rows"
     - Handle malformed rows with on_bad_lines='warn' and count skipped rows -> warning
   - Detect encoding automatically before parsing
   - Return parsed DataFrame if valid

Install dependencies first: `pip install charset-normalizer`
  </action>
