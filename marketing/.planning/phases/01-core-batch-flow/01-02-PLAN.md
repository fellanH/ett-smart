<task type="auto">
  <name>Task 3: Add error handling and rate limit compliance</name>
  <files>webapp/enrichment.py, webapp/app.py</files>
  <action>
Harden the enrichment pipeline:

1. In enrichment.py, add robust error handling:

   ```python
   def enrich_company(company_name: str, org_number: str = None) -> dict:
       result = {
           "company_name": company_name,
           "org_number": org_number or "NOT PROVIDED",
           "status": "pending",
           "allabolag_url": None,
           "fetch_success": False,
           "error_message": None
       }

       try:
           # Generate search URL
           result["allabolag_url"] = generate_allabolag_search(company_name)

           # Fetch with rate limiting built into BatchFetcher
           fetcher = BatchFetcher(delay=1.5, timeout=30)
           fetch_result = fetcher.fetch_url(result["allabolag_url"])

           result["fetch_success"] = fetch_result["success"]
           if fetch_result.get("blocked"):
               result["status"] = "blocked"
               result["error_message"] = "Website blocked automated access"
           elif fetch_result["success"]:
               result["status"] = "success"
           else:
               result["status"] = "error"
               result["error_message"] = fetch_result.get("error", "Unknown error")

       except Exception as e:
           result["status"] = "error"
           result["error_message"] = str(e)

       return result
   ```

2. In enrich_batch, ensure rate limiting:

   ```python
   import time

   def enrich_batch(companies: list, progress_callback=None) -> list:
       results = []
       total = len(companies)

       for i, company in enumerate(companies):
           result = enrich_company(
               company["company_name"],
               company.get("org_number")
           )
           results.append(result)

           if progress_callback:
               progress_callback(i + 1, total)

           # Rate limiting: 1.5s between requests (except last)
           if i < total - 1:
               time.sleep(1.5)

       return results
   ```

3. In app.py, wrap processing in try/except:

   ```python
   try:
       results = enrich_batch(companies, progress_callback=update_progress)
       st.session_state.enrichment_results = results
       st.session_state.workflow_step = "complete"
   except Exception as e:
       st.error(f"Processing failed: {str(e)}")
       st.session_state.workflow_step = "ready_to_process"
   ```

4. Add timeout warning for large batches:
   ```python
   if len(companies) > 20:
       st.warning("Large batch may take several minutes. Consider splitting into smaller batches.")
   ```
