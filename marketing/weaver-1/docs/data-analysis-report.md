Data Enrichment Discussion

The current data only has company names + occupation types. To create an actionable prospect list, we need:

1. Company Location Data

Options:

- Bolagsverket API - Swedish Companies Registration Office (official, has addresses)
- Allabolag.se - Free company lookup (scraping required)
- Proff.se - Business directory with company details

2. Company Turnover/Size

Options:

- Bisnode/Dun & Bradstreet - Comprehensive financial data (paid)
- Allabolag.se - Basic financials (revenue, employees)
- UC/Creditsafe - Credit data providers (paid)

3. Decision Maker Contacts (HR, PM, Management)

Options:

- LinkedIn Sales Navigator - Best for finding contacts (subscription)
- ZoomInfo / Apollo.io - Contact databases (paid)
- Manual research - Company websites, LinkedIn
