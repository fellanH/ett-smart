# ett-smart

API integration project connecting the HubOne apartment platform with the Happy Booking reservation system.

## Structure

```
api/              — Integration API between HubOne and Happy Booking
ett-smart/        — Reference materials and meeting notes
ett-smart-llms-txt/ — LLM-readable API documentation and context
marketing/        — Marketing materials
```

## Stack

Node.js, TypeScript, REST APIs (HubOne, Happy Booking)
